# 大创研究：

欢迎来到我们的小组协作仓库！👋
为了方便大家共享资料和协同工作，以下是简单的操作指南。


##  1. 目录结构说明
为了保持整洁，请大家把文件上传到对应的文件夹中：

* **`A_老师研究`** 
    * 存放目标导师发表的论文、指导过的过往项目、研究生毕业论文等。
* **`B_前沿热点`** 
    * 存放近3年教育领域的热点文献（如数字化转型、STEM教育、AI教育等等）。
* **`C_往年获奖`** 
    * 存放往年教育学领域“大创”的获奖优秀论文
* **`D_政策方向`** 
    * 存放国家关于“教育强国”的政策文件，新闻报道，十五五规划等等

##  2. 基础操作指南 

### 2.1 如何上传文件
1.  点击进入你要上传的**文件夹**（例如点进 `B_前沿热点`）。
2.  在右上角找到 **Add file** 按钮，选择 **Upload files**。
3.  直接把你的文件（PDF/Word等）**拖拽**到灰色区域。
4.  * 等待进度条跑完后，在底部的 "Commit changes" 框里：
    * 第一行写简单的备注（例如：*上传了关于AI教育的3篇综述*）。
    * 点击绿色的 **Commit changes** 按钮保存。

### 2.2 如何下载文件
* **下载单个文件**：
    * 点进文件 -> 点击右上角的 **下载图标** (Download raw file)（但是GitHub比较麻烦，只能一个一个下载🥹，也可以去网上搜索插件批量下载）。
* **下载全部资料**：
    * 回到仓库首页 -> 点击绿色的 **<> Code** 按钮 -> 选择 **Download ZIP**。

### 2.3 如何删除文件
1.  点进你要删除的那个文件（注意：必须先点进文件详情页）。
2.  在右上角找到 **垃圾桶图标** (Delete this file)。
3.  点击后，页面会跳转，直接点击底部的绿色 **Commit changes** 按钮确认删除。
    * *⚠️ 注意：删除操作是不可逆的，操作前请确认没删错！*
  
### 2.4 如何修改文件
* *适用于：修改 README 说明，或者在线编辑 Markdown 笔记。*
1.  点进文件，点击右上角的 **小铅笔图标** (Edit this file)。
2.  在线修改内容。
3.  改完后，点击底部的 **Commit changes** 保存。
